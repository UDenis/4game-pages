# Предметы

> Обладатели Талисмана Вожделения теперь могут улучшить его до&nbsp;Талисмана из&nbsp;Позолоченного Дерева. А&nbsp;любителей крафтить порадует возможность случайным образом получить из&nbsp;сундуков рецепты рангов R/R95/R99.


## Кулоны

1. В&nbsp;связи с&nbsp;введением новых характеристик (LUC, CHA) в&nbsp;игру добавлены кулоны с&nbsp;соответствующими бонусами. 

2. У&nbsp;NPC&nbsp;&mdash; Членов Союза за&nbsp;Восстановление Адена можно обменять вставленные в&nbsp;головные уборы кулоны на&nbsp;Запечатанные Кулоны. 
    _СКРИНШОТ_
    * Запечатанные Кулоны&nbsp;&mdash; предметы, необходимые для создания случайных кулонов. 
    * При обмене на&nbsp;Запечатанный Кулон головной убор исчезает. 

3. Добавлены схемы создания случайных кулонов.
    * Схемы создания случайных кулонов может применять только Мастер Тира.
    * Схему можно с&nbsp;определенной вероятностью получить при использовании Печати Воли.
    * При создании случайного кулона игрок со&nbsp;100%-ной вероятностью получает кулон, дающий бонус к&nbsp;базовым характеристикам персонажа. 

## Рецепты

1. Добавлены случайные рецепты рангов R/R95/R99.
    _СКРИНШОТ_
    * Данные рецепты позволяют получить один предмет снаряжения (оружие/броню/аксессуар), доступного для изготовления на&nbsp;данном уровне.
    * Снаряжение появляется в&nbsp;сундуке. Из&nbsp;него с&nbsp;определенной вероятностью можно получить Благословенный Предмет.
     * Необходимые для создания материалы и&nbsp;рецепты можно получить присвоением в&nbsp;полевых зонах охоты.

2. Добавлены рецепты трех видов, позволяющие разбирать снаряжение рангов R/R95/R99.
    _СКРИНШОТ_
    * Получить рецепты можно в&nbsp;рейдах на&nbsp;боссов (Антарас, Валакас, Линдвиор, Экстремальная Истхина, Земляной Червь, Экстремальный Октавис, Экстремальный Таути). Применять их&nbsp;могут только Мастера Тира.
    * После применения рецепта можно разбирать соответствующие предметы.
    * При разборе вероятности неудачи нет&nbsp;&mdash; игрок гарантированно получает сундук с&nbsp;компонентами предмета.
    * Разбор снаряжения с&nbsp;Душой или снаряжения, дающего особые бонусы, невозможен. Других наград согласно уровню Зачарования не&nbsp;предусмотрено. 


## Талисманы

1. Добавлена функция улучшения Талисмана Вожделения до&nbsp;нового предмета&nbsp;&mdash; Талисмана из&nbsp;Позолоченного Дерева.
    * Создание Талисмана из&nbsp;Позолоченного Дерева возможно путем &laquo;составления&raquo;. 
    _СКРИНШОТ_ 
    * Запечатать Талисман из&nbsp;Позолоченного Дерева, как и&nbsp;Талисман Вожделения, можно у&nbsp;Управляющего Талисманами Семени.
    _СКРИНШОТ_
    * Запечатанный Талисман можно продать или обменять. 

2. Для запечатывания Талисмана Вожделения теперь нужен не&nbsp;Самоцвет: Ранг&nbsp;R, а&nbsp;Сила Гигантов.

3. У&nbsp;торговца украшениями в&nbsp;Беноне продается Талисман Бенира, который можно усилить.
    * Талисман Бенира с&nbsp;определенной вероятностью можно усилить, если объединить 2&nbsp;данных предмета одинакового уровня. 


## Прочие изменения

1. Изменен список материалов, необходимых для обмена на&nbsp;Браслет Агатиона в&nbsp;магазине фирменных вещей &laquo;Александрия&raquo; в&nbsp;Гиране.
    * После смерти персонажа больше не&nbsp;исчезает эффект предметов &laquo;Настойка Всплеска Энергии&raquo; и&nbsp;&laquo;Зелье Поддержания Энергии (30&nbsp;мин.)&raquo;.

2. Иконка Припасов Преторианца теперь отображается корректного.

3. В&nbsp;описание свойств предмета &laquo;Руна Квалификации&raquo; добавлена детальная информация.

4. В&nbsp;описание свойств предмета &laquo;Удочка Героя&raquo; добавлена детальная информация. 

5. В&nbsp;описание предмета &laquo;Книга Ученика&raquo; добавлена информация об&nbsp;эффекте &laquo;Заряд Души&raquo;, возникающем после изменения его внешнего вида.

6. Кастеты, надетые персонажем в&nbsp;Плавательном костюме, отображаются корректно.

7. Исправлены установки физической атаки Парных Кинжалов Линдвиора.

8. Устранена проблема, когда у&nbsp;предмета &laquo;Браслет Арии&raquo; не&nbsp;применялась опция максимального увеличения&nbsp;HP на&nbsp;531.

9. Изменено описание умения &laquo;Дыхание Когтя Дракона&raquo; (оружие мирового босса) высшего уровня.

10. Исправлена ошибка, при которой не&nbsp;применялась задержка повторного применения предмета &laquo;Рыбное Рагу&raquo;.

11. Звук при удалении Печати Агатиона при надевании предмета &laquo;Браслет Призыва Агатиона: Фея&raquo; теперь воспроизводится нормально. 

12. Из&nbsp;Сундука Поддержки Тома удален предмет &laquo;Сундук Поддержки Судьбы&raquo;.

13. Теперь нельзя получить снаряжение Железных Врат с&nbsp;ограниченным периодом использования при обмене Монет Железных Врат.

14. В&nbsp;список оружия, которое можно получить в&nbsp;обмен на&nbsp;Монеты Железных Ворот, добавлены двуручные дубины рангов D&mdash;S.

15. Вес Руды Души изменен на&nbsp;&laquo;0&raquo;.

16. Немного увеличена вероятность успеха при создании магического оружия.

17. Отменены условия для ношения мантий ранга&nbsp;S и&nbsp;выше.
    * При несоответствии уровня персонажа уровню мантии, налагается штраф.

18. Можно отправлять предметы персонажам одной учетной записи через NPC-Алхимика.

19. Добавлена функция обмена предметов с&nbsp;опциями &quot;Кровавый&quot;/&quot;Тьмы&quot; на&nbsp;предметы с&nbsp;опциями &quot;Тьмы&quot;/&quot;Кровавый&quot; у&nbsp;Кузнеца Маммона.

20. Получить Свитки Снятия Оков после смерти рейдовых монстров/боссов теперь нельзя.

21. Снижена цена на&nbsp;покупку в&nbsp;магазинах снаряжения без ранга&nbsp;&mdash; ранга&nbsp;B (оружие, доспехи, аксессуары).

22. Уменьшено количество кристаллов, получаемых после кристаллизации снаряжения рангов D&mdash;B, доступных для покупки в&nbsp;магазинах.

23. Уменьшено количество материалов, необходимых для создания снаряжения рангов D&mdash;B, доступных для покупки в&nbsp;магазинах.

24. Проблема, когда было невозможно изменить внешний вид ряда головных уборов, устранена.

25. Исправлена ошибка, при которой не&nbsp;изменялся цвет волос персонажа после применения Красящего зелья.

26. У&nbsp;NPC-торговцев каждой расы на&nbsp;Говорящем острове теперь можно купить кастеты и&nbsp;двуручные посохи без ранга.

27. Изменен порядок отображения предметов на&nbsp;продажу у&nbsp;NPC. 

28. Появилась возможность добавления бонуса точности к&nbsp;Одноручному Топору Таути, Двуручному Топору Таути, Парным Топора Таути и&nbsp;Топору Таути.

29. Устранена проблема невозможности получения Кодекса Гигантов после убийства определенных NPC. 
    * Испорченный Азоф больше не&nbsp;установлен как необходимый материал для следующих рецептов: &laquo;Эликсир Ментальной Силы (Ранг B)&raquo;, &laquo;Эликсир Ментальной Силы (Ранг A)&raquo;, &laquo;Эликсир Ментальной Силы (Ранг S)&raquo;.

30. Модификации внешнего вида теперь сбрасываются во&nbsp;время обмена оружия или редкой брони у&nbsp;Кузнеца Маммона.

31. Возможно использовать предмет для восстановления энергии без задержки повторного использования. 

32. Обычная Булава Якса при комиссионной продаже больше не&nbsp;распознается как двуручное оружие.

33. Устранена проблема невозможности хранения на&nbsp;Складе Клана Свитка 1&nbsp;000&nbsp;000&nbsp;SP.

34. Исправлена ошибка, при которой предмет нельзя было использовать даже несмотря на&nbsp;то, что все условия удовлетворяются.

35. К&nbsp;эффектам Свитка Ветра Фреи добавлено повышение получаемых SP.

36. После открытия личной лавки свойства и&nbsp;внешний вид товаров отображаются корректно.

37. Шлем Воина больше не&nbsp;отображается на&nbsp;ногах персонажа.

38. При использовании предмета &laquo;Книга Древних&nbsp;&mdash; Божественное Вдохновение&raquo; (нач./ср. уровни) больше не&nbsp;выводится ненужное системное сообщение.

39. Исправлены неточности в&nbsp;информации о&nbsp;количестве получаемых&nbsp;SP в&nbsp;описании ряда SP-свитков.

40. Исправлены неточности в&nbsp;информации о&nbsp;выпадении, обмене, личных лавках и&nbsp;хранении на&nbsp;складе для усиливающих доспехи/оружие Камней Гигантов и&nbsp;Ветра&nbsp;&mdash; ранг R.

41. Снижена цена на&nbsp;продажу снаряжения ранга&nbsp;B в&nbsp;магазинах фирменных вещей.

42. В&nbsp;случае неудачи Зачарования с&nbsp;помощью Камня Модификации Фреи уровень предмета больше не&nbsp;отображается как &laquo;+ 0&raquo;.

43. Увеличена плата за&nbsp;создание Парных Кинжалов ранга A.

44. На&nbsp;продажу и&nbsp;количество кристаллов для ряда оружия ранга&nbsp;B установлена некорректная цена.

45. Информация о&nbsp;времени повторного применения умения &quot;Талисман ветра&nbsp;&mdash; Защита&quot;/&quot;Защита от&nbsp;Злых Чар&quot; теперь отображается корректно. 

46. На&nbsp;мантии клана, надетой вместе с&nbsp;предметом, превращенным в&nbsp;Костюм Деда Мороза, теперь отображается знак клана.

47. Исправлена ошибка, при которой часть головных уборов типа &laquo;Бантик&raquo; занимали только один слот для головных уборов.
    * Если на&nbsp;персонаже надеты Бантик и&nbsp;другой головной убор, то&nbsp;в&nbsp;инвентаре может отобразиться надетым только Бантик, однако если снять головной убор и&nbsp;снова открыть инвентарь, то&nbsp;ситуация нормализуется.

48. Уровень лабиринта персонажа в&nbsp;описании предмета &laquo;Билет в&nbsp;Лабиринт Императора&nbsp;&mdash; Ур. 90/95&raquo; (группа) теперь отображается корректно.

49. Исправлена ошибка, при которой у&nbsp;персонажей-Орков, использующих Музыкальный Инструмент, символ отображался только в&nbsp;правой руке.

50. Теперь отображается детальная информация по&nbsp;параметрам эффектов &laquo;Все характеристики&raquo; и&nbsp;&laquo;Все параметры&raquo;.

51. Название предмета и&nbsp;умения &laquo;Талисман Ветра&nbsp;&mdash; магический/физический критический урон&raquo; изменено на&nbsp;&laquo;Критический урон&raquo;.

52. Треугольный знак на&nbsp;предмете при использовании Камня Обработки, позволяющего полностью изменить его внешний вид, больше не&nbsp;отображается в&nbsp;зеленом цвете.

53. При использовании Свитка SP&nbsp;5000 больше не&nbsp;появляется системное сообщение &laquo;Вы&nbsp;используете Свиток SP&nbsp;50000&raquo;.

54. Скорость персонажа, находящегося под воздействием эффекта увеличения скорости перемещения, на&nbsp;Сияющей Освещенной Сцене не&nbsp;становится слишком высокой.

55. Изменена иконка предмета и&nbsp;умения &laquo;Зелье Удачи Мафр&raquo;.

56. В&nbsp;информации о&nbsp;предмете &laquo;Зелье Удачи Мафр&raquo; больше не&nbsp;отображается время повторного применения.

57. Во&nbsp;время усиления предмета при помощи Свитка Гигантов + 3&nbsp;теперь выводится предупреждающее системное сообщение.

58. Графический эффект &laquo;Зачарование оружия Темный/Кровавый Воитель Хелиоса&raquo; теперь отображается корректно.
    * Исправлена ошибка, при которой ряд головных уборов, надетых персонажами расы Артея, отображался как &laquo;Модная Шляпа&raquo;: &laquo;Модная Шляпа с&nbsp;Перьями&raquo;, &laquo;Элегантная Голубая Модная Шляпа&raquo;, &laquo;Элегантная Зеленая Модная Шляпа&raquo;, &laquo;Элегантная Красная Модная Шляпа&raquo;.
    * Устранена проблема некорректного отображения следующих предметов, надеваемых персонажами расы Артея: &laquo;Модная Шляпа Удачи&raquo;, &laquo;Модная Шляпа Харизмы&raquo;, &laquo;Элегантная Модная Шляпа&raquo;, &laquo;Перчатки Знаний&raquo;, &laquo;Сапоги Знаний&raquo;.

59. Исправлена ошибка, при которой персонажи рас Камаэль и&nbsp;Артея не&nbsp;могли надевать ряд доспехов.

60. Модель персонажа расы Артея, надевшего Тунику или Штаны Ученика, теперь отображается правильно.

61. Элегантная Шляпа и&nbsp;Модная Шляпа больше не&nbsp;отображаются на&nbsp;персонажах расы Артея слишком темными.

62. Персонажи расы Артея теперь могут использовать предмет &laquo;Упаковка: Комплект Снаряжения 90&nbsp;дн.&raquo;

63. Персонажи расы Артея теперь могут использовать Свиток Призыва: Ненавистный Демон.

64. Перчатки/Ботинки Таллума больше не&nbsp;отображаются на&nbsp;персонажах расы Артея как Латные Перчатки/Ботинки Кошмаров.

65. Мантия Кристалла Тьмы больше не&nbsp;отображается на&nbsp;персонажах расы Артея как Мантия Кошмаров.
