# Зоны охоты

> Игроков 99&nbsp;уровня порадует новая зона охоты &laquo;Пространственный Барьер&raquo;. В&nbsp;ряде других зон изменены настройки монстров: теперь они сильнее, а&nbsp;количество получаемого опыта&nbsp;&mdash; больше.

* Добавлена зона охоты &laquo;Пространственный Барьер&raquo;
	_СКРИНШОТ_
    * Войти в&nbsp;зону могут только персонажи 99&nbsp;уровня с&nbsp;помощью Решета в&nbsp;Беноне.
    _СКРИНШОТ_
    * В&nbsp;Пространственном Барьере одновременно может находиться от&nbsp;1&nbsp;до&nbsp;4&nbsp;игроков.
    * Персонажи выводятся из&nbsp;подземелья через 15&nbsp;минут после входа в&nbsp;него, однако поднявшись на&nbsp;более высокий уровень, можно продлить время (с&nbsp;повышением уровня повышается и&nbsp;сложность).
    * Вероятность возникновения событий в&nbsp;подземелье определяется количеством Трансформаций Измерений, которые можно получить у&nbsp;Эйнштера.
    * Вход возможен только при наличии трех и&nbsp;более Трансформаций, которые можно приобрести у&nbsp;Решета и&nbsp;Эйнштера.
    * Всего может произойти 5&nbsp;событий, при которых появляются монстры класса Бес; Можно получить награду высокого уровня.
    * После прохождения 1-го этапа задания &laquo;Исследование Пространственного Барьера&raquo; можно получить Браслет Иного Измерения; с&nbsp;прохождением каждого нового этапа задания браслет будет усиливаться на&nbsp;1&nbsp;уровень. 

* Добавлены зоны охоты для расы Артея: &laquo;Бенон&raquo; и&nbsp;&laquo;Лес Ветров&raquo;.
    _СКРИНШОТ_
    * В&nbsp;Бенон можно попасть с&nbsp;Говорящего Острова, из&nbsp;Глудина и&nbsp;Глудио.
    * В&nbsp;Лесу Ветров можно развить персонажа до&nbsp;20&nbsp;уровня. 
	_СКРИНШОТ_

* Добавлена зона &laquo;Остров Дьявола&raquo;
    _СКРИНШОТ_ 
    * После разрушения Печати Наказания возник Пространственный Разлом, из&nbsp;которого Энергия Иных Измерений просочилась в&nbsp;разные уголки мира.
    * Остров Дьявола изменился под воздействием этой энергии, и&nbsp;корсары и&nbsp;нежить, обитавшие на&nbsp;нем, в&nbsp;одночасье куда-то исчезли.

* Условия использования умения &laquo;Правосудие Иного Измерения&raquo; Архонтом Иного Измерения облегчены.

* Вероятность воскрешения и&nbsp;успеха части монстров в&nbsp;Долине Драконов (ночное время) уменьшена.

* Настройки монстров изменены в&nbsp;следующих зонах охоты:

> | Зона охоты | Изменения |
> | ---------------- | ---------------------------------------------------------------- |
> | Забытые Равнины | Увеличено количество&nbsp;HP монстров и&nbsp;количество получаемого опыта. |
> | Поле Битвы | Увеличено HP&nbsp;всех монстров, кроме Главы Злобных Душ и&nbsp;его слуг;<br>Увеличено количество получаемого опыта. |
> | Болото Криков | Увеличено количество&nbsp;HP монстров и&nbsp;количество получаемого опыта. |
> | Лес Неупокоенных | Увеличено количество&nbsp;HP монстров и&nbsp;количество получаемого опыта. |

* Теперь персонаж не&nbsp;может с&nbsp;помощью умения переместить Рабочего Склада Острова Грез.

* Монстры Леса Неупокоенных больше не&nbsp;могут использовать умение Удержание.

* Исправлена ошибка, при которой Кимериан в&nbsp;Колонии Фей не&nbsp;оставлял предмет для входа в&nbsp;подземелье после убийства Фан Дирема.

* Монстры больше не&nbsp;атакуют персонажа во&nbsp;время использования Рамуна в&nbsp;левой части Перекрестка Разбойников.

* Исправлена ошибка, при которой разговор с&nbsp;Герольдом Невита в&nbsp;Деревне Охотников после успешной атаки Антараса был невозможен.

* Рейдовый босс Копия Хаффа больше не&nbsp;появляется в&nbsp;недоступной для игрока локации в&nbsp;окрестностях Эльфийской Деревни.

* Изменено количество получаемых&nbsp;SP, очков опыта и&nbsp;список выпадающих предметов в&nbsp;Саду Бытия. 

* Произведены следующие изменения подземелья &laquo;Семя Бессмертия&raquo;:
    * Минимальное количество игроков для входа в&nbsp;подземелье изменено на&nbsp;14;
    * Количество появляющихся Дьявольских Колоний Гнилой Плоти изменено с&nbsp;5&nbsp;на 3;
    * Боевые характеристики ряда монстров снижены. 

* Снижены боевые характеристики ряда монстров подземелья &laquo;Семя Разрушения&raquo;.

* Снижено количество&nbsp;HP монстра Слуга Дариона. 

* В&nbsp;список наград за&nbsp;убийство рейдового босса добавлен Камень Жизни Арии.

* Возврат после входа из&nbsp;Грани Реальности в&nbsp;Грань Башни и&nbsp;Грань Площади теперь происходит спустя 1&nbsp;минуту.
    * Статус возврата обновляется каждый день в&nbsp;6-30&nbsp;утра.

* В&nbsp;подземелье &laquo;Рыцари Балтуса: Экспедиция Антараса&raquo; уменьшено количество Панцирных Драконов и&nbsp;исправлена ошибка, при которой они самоуничтожались. 

* Находясь в&nbsp;рейде, игроки, уровень которых на&nbsp;11&nbsp;ниже уровня босса, даже состоя в&nbsp;альянсе, теперь могут получить предметы.

* На&nbsp;Орфена теперь можно наложить злые чары. 

* На&nbsp;финальном этапе прохождения подземелья &laquo;Рыцари Балтуса: Экспедиция Антараса&raquo; теперь не&nbsp;появляются Панцирные Драконы.

* Условия входа и&nbsp;способ прохождения рейдов Белефа и&nbsp;Дариона изменены следующим образом:
    * Рейды появляются раз в&nbsp;7&nbsp;дней;
    * Возможен вход только для одного альянса, состоящего минимум из&nbsp;98&nbsp;игроков;
    * Вход через Червоточину, а&nbsp;также дистанционно невозможен;
    * Удалены все Червоточины Острова Ада, кроме Червоточины возле Деревни Беженцев;

* Стражи Древнего Города Аркана больше не&nbsp;атакуют персонажей с&nbsp;обычной кармой. 

* Удалены некоторые NPC из&nbsp;подземелья Экспедиция Антараса.

* Древний Город Аркан теперь разделен на&nbsp;мирные зоны.

* Исправлена ошибка, при которой в&nbsp;Монастыре Безмолвия невозможно было получить предметы &laquo;Ключ от&nbsp;Сверкающей Комнаты&raquo; и&nbsp;&laquo;Потухший факел&raquo;.

* Алтарь Душ, где можно использовать Призрачный Камень и&nbsp;призвать рейдового босса, теперь находится в&nbsp;окрестностях Бенона.
    * Алтари Душ в&nbsp;окрестностях городов Темных Эльфов, Эльфов и&nbsp;Орков удалены;
    * С&nbsp;помощью Алтаря Душ можно призвать к&nbsp;Бенону рейдового босса уровнем 88/93/98;
    * Рейдовый босс исчезает, если игроки удаляются от&nbsp;центра Алтаря на&nbsp;определенную дистанцию;

* Рейд в&nbsp;специальной зоне Белефа на&nbsp;Острове Ада теперь невозможно начать заново.
    * Разорвав соединение и&nbsp;не&nbsp;переподключившись в&nbsp;течение 20&nbsp;минут, игрок перемещается в&nbsp;ближайший город.

* Исправлена ошибка, при которой локация &laquo;Тень Древа Жизни&raquo; считалась мирной зоной. 

* При попытке воспользоваться Телепортом в&nbsp;Заброшенной Мастерской больше не&nbsp;происходит разрыва соединения.

* Предметы, остающиеся после смерти всех NPC зон охоты, изменены.

* Количество получаемых аден в&nbsp;зонах охоты с&nbsp;1&nbsp;по&nbsp;85&nbsp;уровень изменено.

* Предметы, остающиеся после смерти рейдовых монстров и&nbsp;боссов, изменены.

* Получить Свитки Снятия Оков после смерти рейдовых монстров/боссов теперь нельзя.
* Остается возможность получения Свитков от&nbsp;Членов Союза Восстановления Адена.

* Теперь нельзя присвоить дополнительную награду после убийства монстров зоны охоты &laquo;Остров Ада&raquo;&nbsp;&mdash; для этого добавлен специальный NPC Карвингрид. 

* Сильно ограничено получение Камней Жизни уровнем ниже&nbsp;76, так как их&nbsp;полезность заметно снизилась.

* Теперь на&nbsp;Острове Дьявола не&nbsp;появляются монстры.

* Из&nbsp;зоны охоты &laquo;Кузница Богов&raquo; удален монстр Жук Тар.

* В&nbsp;сундуках с&nbsp;сокровищами в&nbsp;Грани Реальности больше не&nbsp;появляются испорченные материалы.

* У&nbsp;ряда монстров Долины Драконов удалено умение &laquo;Страх&raquo;.

* Рейдовый босс Кровоточащая Муха, появляющийся в&nbsp;Долине Драконов, теперь исчезает, если не&nbsp;оказывается атакован в&nbsp;течение 1&nbsp;минуты.

* Исправлена ошибка, при которой для использования Телепорта Дворянина с&nbsp;помощью Хранителя Портала Астиэль в&nbsp;Беноне требовалось некорректное количество Символов Олимпиады. 

* Информация об&nbsp;умении &laquo;Опасная зона&raquo;, сообщающем о&nbsp;попадании на&nbsp;территорию, где игрок может получить урон или злые чары, теперь отображается корректно. 

* Исправлена ошибка, при которой не&nbsp;отображалась анимация Смотрителя Картии, появляющегося в&nbsp;Лабиринте Картии для группы 95&nbsp;ур.

* Исправлена ошибка, при которой не&nbsp;отображалась анимация Капитана Наемников Турек из&nbsp;Лагеря Орков.

* Устранена проблема невозможности перемещения в&nbsp;ряд локаций зоны охоты &laquo;Алтарь Зла&raquo;.

* Монстры в&nbsp;локациях зоны охоты &laquo;Лес Ветров&raquo; больше не&nbsp;появляются в&nbsp;локациях, недоступных для перемещения. 

* Исправлена ошибка, при которой после разговора с&nbsp;Хранителем Портала Огненного Дракона в&nbsp;Зале Пламени не&nbsp;открывалась дверь.

* Теперь Деревня Кайнак на&nbsp;Радаре отображается как &laquo;Нет сигнала&raquo;.

* Теперь персонажи на&nbsp;первом участке Руин Эсагира не&nbsp;проваливаются в&nbsp;определенных местах.

* Устранена проблема, при которой на&nbsp;атаку игрока не&nbsp;реагировал ряд монстров:
    * Маг Кровавых Могил;
    * Сумеречное Око;
    * Сумеречная Сила.

* При возникновении Растлевающей Тьмы в&nbsp;Древнем Городе Аркане всегда появляются Загадочный Жрец и&nbsp;Ведьма Йона.

* На&nbsp;территории рейда Валакаса все участки отображаются корректно.

* Хранитель Портала Камил из&nbsp;Деревни Кайна показывает Портал правильно.

* Исправлена ошибка, при которой Призраки Чертубы и&nbsp;Грезы Чертубы не&nbsp;применяли на&nbsp;себя умение Сила.

* В&nbsp;случае неудачного рейда на&nbsp;Линдвиора теперь всегда есть возможность получить награду у&nbsp;Дженны в&nbsp;Руне.

* Исправлена ошибка, при которой в&nbsp;подземелье &laquo;Хрустальный Лабиринт&raquo; не&nbsp;возникал Портал.

* Удален рейдовый NPC Гигантский Голем Хаоса в&nbsp;Древней Лаборатории.

* Уменьшена область возвращения рейдового босса Колонии Фей.

* Персонаж больше не&nbsp;умирает от&nbsp;атаки монстров во&nbsp;время использования Рамуна в&nbsp;области Кровавых Туманов.

* Больше не&nbsp;появляются дополнительные монстры рейдовых боссов Кровоточащая Муха и&nbsp;Могучий Взрыватель.

* Исправлена ошибка, при которой нижняя половина Прямоходящего Зверя Сента появлялась ниже уровня земли.
